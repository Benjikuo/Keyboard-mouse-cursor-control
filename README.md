## ‚å®Ô∏è The hotkeys  
### Toggle Features: 
**"Ctrl" + "M": activate the mouse mode / change it back to the keyboard mode**  

### On mouse mode:  
**Movement**  
- "W" or "I": cursor move up  
- "A" or "J": curcor move left  
- "S" or "K": curcor move down  
- "D" or "L": curcor move right

pressing keys above while pressed "Shift" or ";": speed up  
pressing keys above while pressed "Alt": speed down  

**Click**  
- "Q" or "U": curcor move left  
- "E" or "O": curcor move down  
- "Z" or "M": curcor move right   

**Scroll**  
- "X" or ",": scroll up  
- "C" or ".": scroll down

### Disable mouse mode: 
The program will automatically switch back to keyboard mode if the user presses keys that are not defined as hotkeys.
